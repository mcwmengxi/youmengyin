Container 布局容器 #

基本使用 #

用于布局的容器组件，方便快速搭建页面的基本结构：

<qx-container>：外层容器

<qx-header>：顶栏容器

<qx-aside>：侧边栏容器

<qx-main>：主要区域容器

<qx-footer>：底栏容器

常见页面布局







显示代码
<template>
  <qx-container>
    <qx-header>Header</qx-header>
    <qx-main>Main</qx-main>
  </qx-container>

  <qx-container>
    <qx-header>Header</qx-header>
    <qx-main>Main</qx-main>
    <qx-footer>Footer</qx-footer>
  </qx-container>

  <qx-container>
    <qx-aside width="200px">Aside</qx-aside>
    <qx-main>Main</qx-main>
  </qx-container>

  <qx-container>
    <qx-header>Header</qx-header>
    <qx-container>
      <qx-aside width="200px">Aside</qx-aside>
      <qx-main>Main</qx-main>
    </qx-container>
  </qx-container>

  <qx-container>
    <qx-header>Header</qx-header>
    <qx-container>
      <qx-aside width="200px">Aside</qx-aside>
      <qx-container>
        <qx-main>Main</qx-main>
        <qx-footer>Footer</qx-footer>
      </qx-container>
    </qx-container>
  </qx-container>

  <qx-container>
    <qx-aside width="200px">Aside</qx-aside>
    <qx-container>
      <qx-header>Header</qx-header>
      <qx-main>Main</qx-main>
    </qx-container>
  </qx-container>

  <qx-container>
    <qx-aside width="200px">Aside</qx-aside>
    <qx-container>

    </qx-container>
  </qx-container>
</template>
