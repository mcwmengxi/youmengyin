# 安装

## pnpm

使用快速的，节省磁盘空间的包管理工具 [pnpm](https://pnpm.io/zh) 进行安装

```shell
pnpm add ymy-ui
```

## 📦 npm

使用 `npm` 的方式安装

```shell
npm i ymy-ui
```

## yarn

使用 `yarn` 的方式安装

```shell
yarn add ymy-ui
```
